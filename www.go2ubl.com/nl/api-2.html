API documentatie

De technologie voor factuurherkenning van go2UBL is te gebruiken via onze API (Application Programming Interface). Maak gebruik van onze API’s om go2UBL te integreren in jouw oplossing.

Wij bieden momenteel 2 API’s
  • API 1: Deze is bedoelt voor het aanmaken en beheren van klantinstellingen.
  • API 2: Hiermee kan je facturen insturen met jouw eigen ID naar go2UBL en de status van jouw ID volgen. Na het doorlopen van alle statussen kan de UBL+PDF van de factuur door jouw applicatie worden opgehaald bij go2UBL.
Hiernaast kan je de documentatie van de API’s opvragen (inclusief voorbeeld code) en voorbereiden in jouw eigen applicatie. Zodra je de applicatie wilt testen kan je bij ons de credentials opvragen voor zowel de acceptatieomgeving als wel voor de productie omgeving.

Zie voor extra informatie: Node library to use the go2UBL API
