Software-Pakete

go2UBL konvertiert alle Einkaufsbelege in das universelle UBL-Format. Das bedeutet, dass jedes UBL-fähige Softwarepaket für die Kombination mit go2UBL geeignet ist. Nachfolgend finden Sie eine Reihe von UBL-fähigen Paketen, in denen go2UBL bereits erfolgreich eingesetzt wird.


Cloud-Dienstleistungen

Plattformen

Zusätzlich zu den Softwarepaketen verfügt go2UBL auch über eine Anbindung an verschiedene große Plattformen. Diese Plattformen bieten die Möglichkeit, E-Invoices von Maschine zu Maschine zu versenden oder Rechnungen von Lieferanten einzusammeln. Es gibt jedoch immer noch eine große Menge an Rechnungen auf Papier oder im PDF-Format, die go2UBL konvertiert und an diese Plattformen liefern kann.

Darüber hinaus ist go2UBL Mitglied der E-Invoicing-Plattform und trägt ebenfalls das Markenzeichen der elektronischen Rechnungsstellung.
        
    


Handelskammer

go2UBL hat eine direkte "Live"-Verbindung zur Datenbank der Handelskammer.

Über diesen Link wird jede von go2UBL verarbeitete Rechnung auf Anwesenheit bei der Handelskammer überprüft. Darüber hinaus werden die aktuellen Namens-, Adress- und Ortsdaten in das BMG aufgenommen, so dass sie automatisch in die Stammdaten des Gläubigers in Ihrem Softwarepaket aufgenommen werden.

Wenn ein Lieferant bei der Handelskammer abgemeldet wird oder Geisterrechnungen verschickt, wird dies automatisch über go2UBL abgefangen und Sie erhalten automatisch eine E-Mail-Benachrichtigung. Die Rechnung wird aus Sicherheitsgründen nicht zugestellt, so dass sie nicht versehentlich bezahlt werden kann.

API

Die go2UBL-Rechnungserkennungstechnologie kann über unsere API (Application Programming Interface) genutzt werden. Nutzen Sie unsere APIs, um go2UBL in Ihre Lösung zu integrieren. Weitere Informationen finden Sie auf dieser Seite.
