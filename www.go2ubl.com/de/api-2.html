API-Dokumentation

API documentatie

Die go2UBL-Rechnungserkennungstechnologie kann über unsere API (Application Programming Interface) genutzt werden. Nutzen Sie unsere APIs, um go2UBL in Ihre Lösung zu integrieren.

Wir bieten derzeit 2 API’s an
  • API 1: Diese ist für die Erstellung und Verwaltung von Kundeneinstellungen vorgesehen.
  • API 2: Ermöglicht es Ihnen, Rechnungen mit Ihrer eigenen ID an go2UBL zu senden und den Status Ihrer ID zu verfolgen. Nachdem Sie alle Status durchlaufen haben, kann die UBL+PDF der Rechnung durch Ihren Antrag bei go2UBL abgerufen werden.
Darüber hinaus können Sie die Dokumentation der APIs (einschließlich Beispielcode) anfordern und in Ihrer eigenen Anwendung aufbereiten.

Sobald Sie die Anwendung testen möchten, können Sie die Zugangsdaten sowohl für die Akzeptanzumgebung als auch für die Produktionsumgebung anfordern.

Weitere Informationen finden Sie unter: Node library to use the go2UBL API
